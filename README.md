# 3D Models
A bunch of my 3D model files for various projects. They aren't organized particularly well (this is a TODO).

Most of these are for [OpenSCAD](https://openscad.org/). There are also some [blender](https://www.blender.org/) scripts and other things.

Most STLs are not committed here unless they were downloaded from somewhere else (e.g., thingiverse) and then manipulated in OpenSCAD.

Python files are for blender.

Some of the `.scad` files may not work because they rely on files that are elsewhere in my file system and I forgot to add them to this repository.
